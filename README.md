# Polk Web Projesi – İç Mimarlık Web Platformu

Polk Web Projesi, modern iç mimarlık ve tasarım firmaları için geliştirilmiş, modüler ve ölçeklenebilir yapıya sahip bir web arayüzü projesidir.  
Bootstrap tabanlı çok sayfalı mimarisi sayesinde kurumsal firmaların dijital sunum ihtiyaçlarını profesyonel şekilde karşılamak üzere tasarlanmıştır.

## Projenin Amacı

- İç mimarlık ve tasarım firmaları için kurumsal, estetik ve hızlı bir web altyapısı sağlamak  
- Responsive, modüler ve kolay yönetilebilir bir frontend mimarisi oluşturmak  
- Uzun vadede CMS ve backend entegrasyonuna uygun bir temel kurmak  

---

## Özellikler

- Çok sayfalı yapı  
- Mobil uyumlu (responsive) tasarım  
- Ortak navbar ve footer yapısı  
- Modüler HTML–CSS–JS mimarisi  
- Admin panel altyapısı  
- Google Maps entegrasyonu  
- Optimize edilmiş performans  
- Türkçe karakter uyumlu yapı
